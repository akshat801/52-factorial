# Visualising 52!

An interactive single-page site that helps you grasp the absurd magnitude of 52 factorial — the number of ways to arrange a standard deck of 52 playing cards.

## What's inside

- **The Number** — The full 68-digit value with scientific notation
- **Scale** — Logarithmic bar chart comparing 52! to familiar large numbers (atoms on Earth, age of the universe, etc.)
- **Analogies** — Real-world thought experiments that put the number in perspective
- **Shuffle** — An interactive card deck you can shuffle, with each arrangement almost certainly unique in all of history

## Tech

Pure HTML, CSS, and vanilla JavaScript — no dependencies, no build step. Just open `index.html`.

## Run locally

```bash
open index.html
# or
python3 -m http.server
```
